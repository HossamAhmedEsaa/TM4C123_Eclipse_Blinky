## TM4C123 - Eclipse

This blinky project is used as a template.

Environment:

    * OS: Ubuntu (Linux)
    * Board: Tiva C - TM4C123GH6PM
    * Compiler: arm-none-eabi-gcc/ar/as/ld
    * Debug: arm-none-eabi-gdb + OpenOCD
    * Tool for debug: Eclipse
